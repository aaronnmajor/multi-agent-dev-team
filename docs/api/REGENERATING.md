# Regenerating the API reference

The HTML files in this directory are generated by `pdoc` from the project's
public modules. To rebuild after a code change:

```bash
pip install pdoc
python -m pdoc -o docs/api agents tools orchestration observability resilience caching memory exceptions config
```

Open `docs/api/index.html` in a browser to browse the result. Two warnings
about `pydantic.main.BaseModel.__pydantic_extra__` and `__pydantic_private__`
type annotations are harmless — they're about Pydantic's internals, not the
project's own code.
